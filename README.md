# Hotel & Event Management System

A full-stack Hotel & Event Management System built using React and Django.

## Features

* Hotel Booking
* Room Management
* Event Management
* User Reservations
* Admin Controls

## Tech Stack

### Frontend

* React.js
* Vite
* CSS

### Backend

* Django
* Django REST Framework

### Database

* SQLite

## Installation

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
pip install -r requirements.txt
python manage.py runserver
```
